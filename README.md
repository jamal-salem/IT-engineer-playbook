# IT Engineer Playbook

A practical reference for IT Engineers focused on:
- Networking
- Automation
- Systems (Windows & Linux)
- Python & PowerShell
- Real-world troubleshooting

This is not a course.
This is a working engineer's notebook.


## How to use this playbook

- Each section focuses on real operational tasks.
- Commands are documented with context and use cases.
- Scripts are written to solve actual IT problems.
- This repository grows with experience, not theory.

## Roadmap

- Networking fundamentals & troubleshooting
- Python automation for systems & networks
- PowerShell administration & automation
- Linux internals & system behavior
- Real-world IT scenarios


## Repository Structure

Each top-level directory represents a core IT domain.
Inside each domain, content is organized into commands, concepts, scripts, and real-world scenarios.
